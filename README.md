## 🎨 Design Philosophy: "The Spatial Archive"

The website's visual identity is built on a custom design system that reflects our architectural roots:
- **Stark Contrast:** High-contrast structural foundation using deep surfaces and sharp linen breaks.
- **Brutalist Geometry:** Strictly sharp corners (0px radius) across all UI components.
- **Luminous Atmosphere:** Depth is created through massive, blurred ambient light orbs rather than drop shadows.
- **Blueprint Typography:** A high-contrast pairing of museum-grade serifs for display and raw, technical monospaced fonts for data and labels.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set the `GEMINI_API_KEY` in `.env.local`:
   ```bash
   GEMINI_API_KEY=your_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
